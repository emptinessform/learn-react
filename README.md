# learn-react

React를 직접 만들면서 배우고, 동시에 다른 사람을 가르칠 수 있는 **React 강의 자료
사이트**입니다. 강의 자료 사이트 자체를 React로 구현하는 과정이 곧 React 학습이
됩니다.

## 무엇을 담나요

- **커리큘럼**: 기초부터 실전까지, 각 강의는 설명 + 코드 + 편집·실행 가능한
  플레이그라운드를 함께 제공
- **진도 추적**: 강의별 완료 체크 + 전체 진행률
- **지식 베이스** (한 곳에서 계속 누적)
  - Q&A 모음 — 학습 중 나온 질문과 답변
  - 용어 사전 — 용어 정의와 상호 링크
  - 배경과 원리 — JavaScript / HTML / CSS / React의 등장 배경과 동작 원리를
    수제 SVG 도식과 함께 설명

## 기술 스택

Vite · React · TypeScript · React Router · react-live · 수제 SVG 도식
