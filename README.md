# learn-react

React를 직접 만들면서 배우고, 동시에 다른 사람을 가르칠 수 있는 **React 강의 자료
사이트**입니다. 강의 자료 사이트 자체를 React로 구현하는 과정이 곧 React 학습이
됩니다.

## ▶ 지금 바로 실행해 보기

**라이브 데모: https://emptinessform.github.io/learn-react/**

설치 없이 위 링크에서 바로 학습할 수 있습니다. (각 강의의 플레이그라운드에서 코드를
직접 고쳐 결과를 즉시 확인하세요.)

## 무엇을 담나요

- **커리큘럼 (19강, 기초→실전)**: 입문 · 컴포넌트 기초 · 상태와 Hooks · 구조화 ·
  실전 · 캡스톤. 각 강의는 설명 + 코드 + 편집·실행 가능한 플레이그라운드를 함께 제공
- **연습문제 · 퀴즈**: 강의마다 실습 과제와 객관식 자가 점검(정답·해설)
- **진도 추적 · 학습 메모 · 백업**: 강의별 완료 체크, 메모(localStorage), JSON 내보내기/가져오기
- **검색**: 강의 · 용어 · Q&A 통합 검색(사이드바 + 키워드)
- **지식 베이스** (한 곳에서 계속 누적)
  - 배경과 원리 — JavaScript / HTML / CSS / React의 배경·원리를 수제 SVG 도식과 함께 설명
  - 용어 사전 · Q&A 모음 · 강의별 "더 읽어보기" 공식 문서 링크
- **다크/라이트 테마 · 모바일 반응형**

## 기술 스택

Vite · React · TypeScript · React Router · react-live · prism-react-renderer · 수제 SVG 도식

## 로컬에서 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm test         # 테스트
npm run build    # dist/ 로 정적 빌드
```

`main`에 푸시하면 GitHub Actions가 자동으로 빌드·배포합니다.
